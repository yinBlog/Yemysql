# MySQL 客户端 (Electron)

一个基于 Electron + mysql2 的桌面版 MySQL 连接管理与查询工具。深色主题，全中文界面。

## 功能

### 多数据库类型
- 连接时可选 **MySQL / MariaDB / PostgreSQL / SQLite**（驱动抽象层 `src/main/drivers.js`，纯 JS 驱动 mysql2 / pg / sql.js，无需本地编译）。
- 选择类型时自动切换字段：服务器型显示主机/端口/用户/密码，**SQLite 显示数据库文件选择（可浏览选 .db/.sqlite，不存在则新建）**；默认端口/用户名也随类型切换。状态栏显示当前连接的类型与版本。
- SQLite 基于 sql.js（WASM）：整库载入内存，写操作后自动持久化回文件；支持浏览/查询/数据增删改/结构查看/导入。
- 跨类型通用：连接、列库/表/字段、运行 SQL、分页浏览（排序/WHERE/行预览）、**数据增删改**（按方言生成占位符与标识符引用）、表结构查看、结构对比。
- PostgreSQL 说明：一个连接对应一个数据库，树中的「数据库」层映射为该库内的 **schema**。可视化的高级 DDL（改字段/索引/外键、SQL dump）目前仅 MySQL/MariaDB 支持，PostgreSQL 下这些入口会自动隐藏或提示用 SQL 操作。

### 连接管理
- 新建 / 编辑 / 删除多个连接，配置本地持久化（系统 userData 目录）。
- 保存前可一键**测试连接**，并显示服务器版本。
- 支持 **SSL** 连接。
- **分组与收藏**：连接可归入自定义分组（编辑时填写，带历史候选），侧边栏按分组折叠展示；标记 ⭐ 收藏的连接置顶于「★ 收藏」组（右键可快速切换）。分组折叠状态本地记忆。
- **密码加密存储**：密码通过 Electron `safeStorage` 加密后落盘（Windows 走 DPAPI，绑定当前系统账户），明文不进入配置文件、也不下发给界面层；连接时由主进程解密。编辑连接时密码框留空表示「不修改」。

### 导出 SQL Dump
- 右键**表** → 导出为 SQL；右键**数据库** → 导出整库为 SQL。
- 可勾选：表结构 (CREATE TABLE) / 数据 (INSERT) / DROP TABLE IF EXISTS。
- 生成标准 dump（`SET FOREIGN_KEY_CHECKS`、分批 INSERT、值安全转义），保存到原生文件对话框选择的位置。

### 主题
- 侧边栏 🌙/☀️ 一键切换**深色 / 浅色**主题，选择记忆在本地。

### 对象树（侧边栏）
- 连接 → 数据库 → 表 三级树，按需展开、懒加载。
- 点击连接自动建连；点击表直接浏览数据。
- **右键菜单**：
  - 连接：连接/断开、刷新、编辑、删除。
  - 表：浏览数据、查看结构、复制建表语句、复制表名、SELECT 到查询、**生成 SELECT / INSERT / UPDATE 模板**、**清空(TRUNCATE)**、**删除(DROP)**。
  - 危险操作（清空/删除）会弹出**原生确认框**二次确认。

### 查询
- 编写并执行任意 SQL，`Ctrl+Enter` 快捷执行。
- **智能执行**：有选中文本时只执行**选中片段**；未选中且编辑器内含多条 `;` 分隔语句时，只执行**光标所在的那条语句**（分号识别会跳过字符串/反引号/注释中的分号），状态栏标注「当前语句」。
- **结果分页（翻页）**：工具栏「每页 ___ 行」（默认 1000，本地记忆）。执行**单条 `SELECT`/`WITH`** 时自动分页——结果网格右上出现 `«  ‹ 上一页  1–20 / 57  下一页 ›  »` 翻页器，翻页即时切换、避免一次拉回海量数据卡死界面（底层用子查询包裹分页 + 计数）。`0` 表示不分页（一次拉全部）；已带 `LIMIT`、多语句、非 SELECT 自动跳过；无法包裹的查询（如结果列名重复）自动回退为普通限制。
- **导出即全部数据**：结果页「复制 / 导出 CSV / 导出 JSON」在分页时按钮标注「（全部）」，会**重新拉取完整结果集**导出，而非仅当前页；「数据」页导出同理，导出符合当前 `WHERE`/排序的整表数据。
- **常驻结构树**：查询页左侧一栏显示当前库的**表 → 字段**（懒加载、🔑 标主键、字段带类型），**单击表/字段名即插入到光标处**，**双击表**生成 `SELECT * FROM 表 LIMIT 100`；顶部搜索框可过滤表/字段；可拖拽调宽、`📚` 按钮随时折叠。记不住表名列名时不用再翻结构页。
- **多查询标签页**：`＋` 新建，可并行保留多份 SQL、数据库选择与结果（含各自翻页状态），互不干扰、可关闭。
- **上下文感知自动补全**：输入时自动提示**表名 / 字段名 / 关键字**（来自当前连接所选库），候选项**带字段类型与 🔑 主键标记**，并按所处子句智能排序——`FROM`/`JOIN` 后优先表名，`SELECT`/`WHERE`/`ON`/`GROUP BY` 等处优先字段名；输入 **`别名.` 或 `表名.`** 时只提示该表的字段（自动解析 `FROM 表 别名` 定义的别名）。`↑↓` 选择、`Enter`/`Tab` 接受、`Esc` 关闭。
- **占位符跳转**：片段/模板插入后，`Tab` / `Shift+Tab` 在 `«占位符»` 之间跳转并选中，输入即替换，快速填空。
- **行注释**：`Ctrl+/`（或「注释」按钮）一键为选中行/当前行添加或取消 `--` 注释。
- **可视化查询构建器**（🧩 构建器）：不用手写 SQL —— 选数据库/表 → 勾选要的字段 → 「＋ 加条件」用「字段 / 运算符（= <> > LIKE IN IS NULL…）/ 值」拼 WHERE → 选排序与 LIMIT，底部**实时生成 SQL**（自动按方言引用标识符、数字不加引号），可一键复制 / 载入编辑器 / 直接执行。
- **SQL 片段库**（📋 片段）：可折叠面板，按「查询 / 修改 / 结构」分组列出常用语句骨架（各种 JOIN、GROUP BY、UPSERT、CREATE TABLE…），点击**插入到光标处**并自动选中第一个 `«占位符»`，输入即替换。
- **右键表一键生成语句**：右键表菜单可生成把**所有字段都列好**的 `SELECT` / `INSERT`（自动跳过自增列）/ `UPDATE`（自动用主键做 `WHERE`）模板，占位符附带字段类型提示，免去手敲字段名。
- 自动区分结果集 / 影响行数，显示耗时。
- **SQL 美化**按钮（轻量格式化）。

### 结果网格（查询结果 / 数据浏览 / 表结构通用）
- **点击表头排序**（查询结果客户端排序，数据浏览服务端排序）。
- **快速过滤**框，实时过滤当前结果。
- **点击单元格**复制其值；**复制**按钮复制整张表（TSV）。
- **导出 CSV / JSON**（原生保存对话框，CSV 带 UTF-8 BOM 防 Excel 乱码）。
- **📊 图表**：一键把当前结果绘成**柱状图 / 折线图 / 饼图**（自选 X / Y 列，纯 SVG 渲染、跟随主题、悬浮显示数值）。
- **双击行**查看行详情（字段竖排，长文本友好）。
- 行号列、NULL 高亮、大结果集渲染上限保护（5000 行）。

### 表结构
- 分段查看：**字段** / **索引** / **外键** / **建表语句**。
- 在左侧点选某张表后，「表结构」标签会**自动同步显示该表**。
- **字段可编辑**（ALTER TABLE）：
  - **＋ 新增字段**：填名称/类型/可空/默认值/注释，并可指定插入位置（最前 / 某字段之后 / 末尾）。
  - **编辑字段**：修改定义（含重命名，底层用 `CHANGE`）。
  - **删除字段**：原生确认框二次确认后 `DROP COLUMN`。
  - 默认值支持「表达式」开关：勾选则原样写入（如 `CURRENT_TIMESTAMP`），否则按字符串加引号。
  - **类型为下拉选择**（可选常用类型 INT/VARCHAR(255)/DATETIME/JSON… 也可手输自定义长度）。
- **索引可视化增删**：
  - 「索引」分段按索引名分组展示（含字段、唯一/普通、🔑 主键、索引方式）。
  - **＋ 新增索引 / 主键**：选类型（普通 / 唯一 / 全文 / 主键）、填名称、多选字段。
  - **删除索引 / 主键**：原生确认框后执行 `DROP INDEX` / `DROP PRIMARY KEY`。
- **外键可视化增删**：
  - 「外键」分段展示约束名、本表字段、引用表(字段)、ON DELETE / ON UPDATE。
  - **＋ 新增外键**：选本表字段、引用表与引用字段、级联动作，生成 `ADD FOREIGN KEY ... REFERENCES`。
  - **删除外键**：确认后 `DROP FOREIGN KEY`。

### 数据导入（与导出对称）
- 入口：数据页工具栏「⭳ 导入」或右键表「导入数据」。
- 支持 **CSV / Excel(.xlsx) / JSON / SQL 脚本** 四种文件。
- **可视化字段映射**：自动按列名匹配「目标字段 ← 来源列」，可逐项调整或设为「不导入」。
- **冲突处理**：追加（直接插入）/ 跳过重复 / 覆盖重复 —— 按方言生成（MySQL `INSERT IGNORE` / `ON DUPLICATE KEY UPDATE`，PostgreSQL `ON CONFLICT`，SQLite `INSERT OR IGNORE/REPLACE`）。
- **进度条 + 事务控制**：整批包在一个事务里，实时进度回传；任一行出错则**整体回滚**，不会写入半截数据。
- SQL 脚本文件则直接执行（忽略映射/冲突设置）。

### 数据库结构对比
- 侧边栏 **⇄** 按钮打开「结构对比」：选**源**库与**目标**库（可跨连接）。
- 对比后给出差异摘要（源独有表 / 目标独有表 / 差异表及字段变化），并自动生成把**目标向源对齐**的**同步 SQL**（`CREATE` / `ALTER ADD|MODIFY|DROP COLUMN`，多余表以注释形式给出 `DROP`）。
- 同步 SQL 仅供审阅，可一键复制或载入到查询编辑器后再执行（不自动执行）。

### 数据浏览与编辑
- 选库选表，支持 **WHERE 条件**过滤、分页（每页 200 行）、表头排序。
- **可编辑数据**（表有主键时）：
  - **双击单元格**直接改值，回车提交（基于主键安全定位，`Esc` 取消）。
  - **右键单元格**：复制 / 编辑 / 设为 NULL / 查看行详情。
  - **点击行号**选中行，「删除选中」批量删除（原生确认框）。
  - **＋ 新增行**：按列填表插入，留空用默认值/自增，可勾选 NULL。
  - 无主键的表自动转为只读并给出提示，避免误伤。
- **行预览面板**：数据页默认在右侧带预览面板，**点击任意行**即在面板中以「字段：值」竖排展示该行完整内容（长文本友好），点击行实时跟随并高亮当前行；可「JSON」一键复制该行、可关闭，工具栏「👁 预览」随时开关（查询结果页也可手动开启）。

### SQL 历史
- 执行过的 SQL 自动持久化（去重、上限 200 条）。
- 历史面板可搜索、点击载入到查询编辑器、一键清空。

### 其他
- 可拖拽调节侧边栏宽度。
- 底部状态栏显示连接信息、服务器版本、操作提示。
- `Esc` 关闭弹窗 / 菜单。

## 运行

```bash
npm install      # 首次安装依赖（会下载 Electron）
npm start        # 启动应用
npm run dev      # 启动并打开开发者工具
npm run smoke    # 无头冒烟测试：加载真实界面，校验渲染与 IPC 无报错
npm run pack     # 打包为免安装目录（release/win-unpacked/）
npm run dist     # 打包为 Windows 安装包（release/MySQLClient-Setup-<版本>.exe）
```

## 打包分发

已配置 **electron-builder**（NSIS 安装包）。`npm run dist` 产出：

- `release/MySQLClient-Setup-1.0.0.exe` —— 可分发的 Windows 安装程序（约 77 MB），支持选择安装目录、创建桌面快捷方式（快捷方式名「MySQL客户端」）。
- `release/win-unpacked/` —— 免安装版，直接运行其中的 `MySQL Client.exe` 即可。

说明：
- `build.electronDist` 指向本地已解压的 Electron，避免重复下载。
- **图标**：`npm run icon`（`scripts/gen-icon.js`）会用纯 JS 生成 `build/icon.ico`、`build/icon.png` 与 `src/icon.png`（蓝色圆角 + 白色数据库圆柱）。安装包图标由 NSIS 直接嵌入；运行时窗口/任务栏图标由主进程 `BrowserWindow.icon` 使用 `src/icon.png`。
- 默认**关闭可执行文件签名/资源编辑**（`win.signAndEditExecutable: false`）。原因：electron-builder 编辑 exe 资源需要 `winCodeSign`，其压缩包含 macOS 符号链接，Windows 非开发者模式下无法解压。若你开启了「开发者模式」或配置了证书，移除该项即可让 exe 自身与快捷方式也带上自定义图标。
- 首次 `dist` 会从 GitHub 下载 NSIS 工具（约 2 MB）到 `.eb-cache/`，之后复用。

## 项目结构

```
src/
  main/main.js        # 主进程：窗口、IPC、mysql2 连接、导出、历史
  preload/preload.js  # 安全 IPC 桥接（contextIsolation）
  renderer/
    index.html        # 界面结构
    styles.css        # 深色主题样式
    renderer.js       # 前端逻辑（对象树、结果视图、历史、右键菜单等）
scripts/smoke.js      # 冒烟测试脚本
```

## 安全说明

- 启用 `contextIsolation`、关闭 `nodeIntegration`，渲染进程仅通过 preload 白名单 API 与主进程通信。
- 库名 / 表名做反引号转义，查询使用参数化占位，规避注入。
- 连接密码经 `safeStorage` 加密后落盘（Windows DPAPI，绑定系统账户），明文不写入配置文件、也不传给渲染进程。旧版明文配置会在下次保存时自动迁移为密文。
